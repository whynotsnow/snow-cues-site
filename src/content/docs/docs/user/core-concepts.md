title: 核心概念
description: 理解空间、空间主密码、关键密钥、规则链和记忆提示。
---

Snow Cues 的使用体验围绕几个核心概念展开。

## 空间

空间是本地数据分区。不同空间拥有不同的空间主密码、规则链、密码条目和迁移关系。

## 空间主密码

空间主密码用于建立当前浏览器内存中的空间会话。它不会被保存，也不会写入导出文件。

## 关键密钥

关键密钥是每条密码单独输入的秘密。新建密码和解密密码都需要它。

## 规则链

规则链决定如何从输入材料生成核心密码。新空间创建密码前需要先初始化规则链。

## 记忆提示

记忆提示是加密保存的辅助提示，不参与密码生成，不参与解密，也不能自动恢复关键密钥。

## 输出适配

输出适配只在核心密码解密后临时应用，用于满足某些平台的密码格式要求。它不会覆盖已经保存的核心密码。
