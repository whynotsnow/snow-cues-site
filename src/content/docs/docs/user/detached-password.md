---
title: 游离密码
description: 空间外的临时密码预览工具，它如何工作，以及如何迁入空间。
---

游离密码是系统工具中的空间外临时预览功能。它让你在不进入空间、不写入 `storageData` 的情况下，临时预览一段密码。

## 基本性质

游离密码只作为当前页面内存中的临时预览。它**不会**：

- 持久化派生密钥。
- 持久化游离规则输入。
- 持久化预览结果。
- 持久化生成历史或可重建材料。
- 写入 `storageData`、localStorage、sessionStorage、URL。
- 写入迁移队列。

预览使用的派生密钥和默认内置规则链都是临时的，只存在于当前页面内存中。

## 使用

游离密码工具只包含派生密钥、核心编码、核心材料长度和声明式输出策略，不包含平台、备注、密码组或记忆提示等正式条目元信息。

你可以用它临时预览一段密码，并决定预览时是否应用声明式输出策略。

## 迁入空间

游离密码的「迁移到空间」是用户交互语义，不使用迁移批次或迁移条目，也不创建空间关系。

- 点击「迁移到空间」后，只保留内存中的待迁入派生密钥草稿。
- 进入目标空间后，你**必须填写**平台、密码组、普通备注和可选记忆提示。
- 可沿用或修改带入的关键密钥。
- 进入目标空间后，按该空间的规则链正式生成并保存条目数据。

:::note
游离密码迁入空间不修改任何来源空间状态，也不创建接替关系。
:::

## 系统工具页

游离密码位于「系统工具」页面，该页面还承载比较两个存储数据文件的工具。这两个工具都不写入 `storageData`，不保存临时输入。
