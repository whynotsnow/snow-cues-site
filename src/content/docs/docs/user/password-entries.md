---
title: 密码条目
description: 新建、解密、编辑、查看提示和废弃密码条目。
---

密码条目保存的是加密后的核心密码输出，以及平台、普通备注、密码组等管理信息。

## 新建密码

新建密码时需要输入关键密钥。关键密钥只参与本次生成和加密，不会保存。

## 解密密码

解密时需要再次输入关键密钥。如果解密失败，请检查空间主密码和该条目的关键密钥。

## 编辑条目

空间校验完成后，可以编辑平台、普通备注、密码组和记忆提示。当前版本不支持直接编辑已保存的密码内容。

## 废弃条目

废弃表示该条记录进入历史状态，不会删除数据。适合平台密码已经重置或不再使用的情况。
