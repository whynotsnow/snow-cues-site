## 应用内指引解决什么

应用内指引会根据当前状态提示可能的下一步，例如：

- 需要先创建或打开 `storageData`。
- 需要进入空间。
- 需要设置空间主密码。
- 需要初始化规则链。
- 需要保存当前草稿。
- 迁移队列还有未完成步骤。
- 当前问题需要阅读更完整的文档说明。

这些提示用于降低遗漏步骤的概率，不代表应用已经替你完成安全判断、备份确认或外部同步状态检查。
