---

密码组用于归类同一平台或系统下的多个账号，并可保存默认输出适配策略。

## 密码组

密码组属于空间内明文管理元数据，允许保存组名、普通说明和声明式输出适配策略。

- 用于管理同一平台或系统下的多个账号。
- 组名、平台和备注都视为隐私元数据，**不要填写账号、关键密钥或敏感规则**。
- 密码组只是 UI 归属，**不参与核心密码生成、解密或输出适配的密码学过程**。

## 输出适配

密码输出适配只在核心密码解密后临时应用。系统用公开、确定性的内置算法，按常见策略或自定义参数生成临时适配密码。

核心密码仍由规则链生成并加密保存，适配策略公开、可审计，不作为安全秘密。

:::important
输出适配**不覆盖**已保存的核心密码密文（`encrypted_password`）。它只在你查看密码时临时转换格式。
:::

## 内置预设

内置策略只提供常见密码策略预设（如长度、字符集要求），**不提供主流平台官方策略**。

- 选择内置预设时，策略字段会锁定。
- 切换到「自定义当前策略」后，才允许基于当前参数调整。

## 自定义策略

自定义策略只能是声明式参数，不能执行用户代码、正则脚本或远程规则。

输出编码支持以下模式：

- **Base62**
- **Base64**
- **自定义字符集**（会去重，且至少需要两个不同字符）

默认输出长度是 24，可在 8 到 64 之间调整。

:::note
内置预设是常见策略，不代表具体平台官方规则。请按目标平台的实际要求选择或自定义。
:::
