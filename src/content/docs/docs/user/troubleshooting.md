## 忘记空间主密码怎么办？

Snow Cues 不保存空间主密码，也没有后端恢复服务。如果忘记空间主密码，系统无法替你恢复。

## 忘记某条密码的关键密钥怎么办？

可以先查看该条目的记忆提示。如果仍无法想起关键密钥，需要去外部平台重置密码，并将旧条目标记为废弃。

## 为什么进入空间后还要校验？

Snow Cues 不保存校验材料。已有密码的空间进入后，需要通过解密一条已知密码来确认当前会话能力。

## 换浏览器后为什么看不到空间？

空间数据保存在当前浏览器本地。换浏览器、换设备或清理数据后，需要通过导出备份恢复。
