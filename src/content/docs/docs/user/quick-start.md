这篇文档用于完成第一次使用流程。

## 第一次使用

1. 打开 Snow Cues 应用。
2. 在空间工作台点击“新建空间”。
3. 选择空白创建，并输入空间标识。
4. 进入空间主页后设置空间主密码。
5. 前往规则管理，初始化规则链。
6. 前往密码管理，新建第一条密码。
7. 填写平台、普通备注、关键密钥和可选记忆提示。
8. 保存后，使用“解密”按钮输入关键密钥查看密码。

## 需要记住什么

- 空间主密码不会保存。
- 关键密钥不会保存。
- 记忆提示会加密保存，但它不能自动恢复关键密钥。
- 清理浏览器数据或换设备前，应先导出备份。
