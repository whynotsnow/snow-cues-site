欢迎来到 Snow Cues 用户文档。

Snow Cues 是一个纯浏览器运行、本地优先的密码派生与加密存储系统。它不提供后端账号、云端同步或自动恢复服务；你需要自己管理空间主密码、关键密钥和导出备份。

官网只分发静态前端代码。你的 `storageData`、空间主密码、关键密钥和密码内容，全部留在你自己手里。

这里的文档面向最终用户，重点解释如何使用、如何理解风险、如何迁移和备份。Snow Cues 的安全边界建立在少保存、显式保存和浏览器原生 WebCrypto 上——文档会明确说明系统保存什么、不保存什么。
